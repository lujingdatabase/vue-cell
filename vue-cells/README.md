# vue-cells

> 常用的组件

## Build Setup

``` bash
# 安装
npm install vue-cells --save

#引用
import vueCells from 'vue-cells'
Vue.use(vueCells)

# 测试环境运行
npm run dev

# 打包发布
npm run build
``

# 注意：
开发与打包发布的入口文件需要切换

For detailed explanation on how things work, consult the [docs for vue-loader](http://vuejs.github.io/vue-loader).
